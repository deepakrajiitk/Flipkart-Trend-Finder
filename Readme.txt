Requirements 
	- Python 3 
	- Pandas
	- Numpy 
	- Streamlit 
	- GoogleSearch 
	- Pytrends
	- Json
	- os 
	- Url lib3
	- serpapi (google-search-results)

RUN
  - To run move into "flipkart-grid/Source-Code/" folder. Open terminal then run command "streamlit run main.py"
  - Website will open in browser.
  - Choose appropriate catergory and subcategory to view results
  - Results are cached to fetch new results click "Re-New" button to refresh results.
